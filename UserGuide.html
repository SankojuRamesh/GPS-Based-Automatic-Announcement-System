1. System Overview

The Train Info GPS Display is an in-vehicle device that shows your train's upcoming stations in real time using GPS. It announces each station name through a built-in speaker and displays the distance remaining to the next stop.

FeatureDescription
Display3.5" colour touch screen, 480 × 320 pixels
GPSLive location tracking — shows distance to each station
AudioStation name announced automatically as train approaches
Trip DataStored on SD card as JSON files, one per route
WiFiDownloads trip data and audio files over the air
Battery3-cell LiPo with live percentage shown in header
VolumeAdjustable 0 – 100% from the Settings menu

3. Getting Started

  1. Insert a FAT32-formatted SD card containing your trip JSON files.
  2. Power on the device. The MindCoin splash screen appears while it initialises GPS, SD, WiFi and battery.
  3. Wait for the progress bar to reach 100% — the Home Menu is shown automatically.
  4. Connect to WiFi once (Settings → WiFi Setting) so trip and audio files can be downloaded.
  5. Tap Trip on the Home Menu, type your train number, then tap SEARCH.
  6. Tap START on the Trip Details screen to begin tracking.
Tip: The device remembers your WiFi credentials, volume level, and all settings between power cycles.

4.10 Approach Distance

When the GPS distance to the next station falls below this value, the display automatically advances to show the following station. Default 500 m, range 100 – 2000 m, step 100 m. Tap BACK to save.

4.11 GPS Match Distance

Used once when a trip is loaded. If the nearest station is within this radius, it becomes the current station (already passed). If farther away, the nearest becomes the next station. Default 200 m, range 50 – 1000 m, step 50 m. Tap BACK to save.

5. Audio Announcements

The system announces each station name automatically as the train approaches.

How it works

  1. GPS reports a new location fix.
  2. Distance to the next station is calculated.
  3. When distance enters the Far–Near zone, the first announcement fires immediately.
  4. Subsequent repeats fire every Gap seconds until Repeats count is reached.
  5. Once all repeats are done, the sequence resets for the following station.

Volume Scaling

Audio volume is applied in software: each 16-bit PCM sample is multiplied by Volume% / 100 before being sent to the I2S amplifier. This means the volume setting has an immediate, accurate effect on speaker loudness.

Announcement Zone Diagram

Train direction →

───────────────[FAR]────────[NEAR]──[STATION]
                 ↑ announcements ↑ no new start

Tip: If announcements repeat too many times, reduce Repeats to 1 or 2 in Announcement Settings.
Tip: If the announcement fires too early, reduce the Far distance.

6. SD Card & Audio Files

FileLocation on SDDescription
Trip data/12345.jsonStation list for train 12345
Audio file/ABC.wavWAV announcement for station code ABC
  • SD card must be FAT32 formatted
  • Audio files are standard WAV (PCM, 16-bit recommended)
  • Missing audio files are downloaded automatically when the station is announced, provided WiFi is connected
  • Use UPDATE on the Trip Details screen to bulk-download all audio for a route
Warning: Do not remove the SD card while the device is powered on.

7. WiFi & Downloads

  • Trip JSON files and audio WAV files are downloaded from the MindCoin API server
  • HTTPS is used — no special certificate setup required
  • If WiFi drops mid-download, the partially saved file is discarded and the operation fails gracefully
  • The device reconnects to WiFi automatically every 5 seconds if the connection is lost
